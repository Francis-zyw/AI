第二步结果人工修订工具

这个文件夹可以直接发给产品经理使用。

使用方法：
1. 双击“打开人工修订工具.bat”。
2. 如果工具目录下的“待修订输入”文件夹里已经放了 JSON 文件：
   - 优先自动挂接 step2_manual_review_package.json
   - 如果没有修订包，则自动挂接 component_matching_result.json
   - 同时会自动挂接 synonym_library.json、run_summary.json（如果存在）
   - 这种情况下不需要手动再选文件，打开后即可直接开始修订
3. 如果“待修订输入”文件夹里没有可识别的 JSON 文件，工具会以空白页面打开，这时再手动选择文件即可。
4. 页面里可以继续修改匹配总结果或源构件词库。
5. 导出时点击：
   - “导出修订包 JSON”：推荐，适合回传给开发侧回写
   - “导出最终匹配结果 JSON”
   - “导出最终源构件词库 JSON”
   - “导出最终摘要 JSON”

说明：
- 工具不依赖 Python，不需要安装任何环境。
- 已修复页面脚本报错，页面可正常加载。
- 已修复 BAT 打开方式，支持双击直接打开工具。
- 自动挂接使用的是 BAT 生成的同目录临时打开页“第二步结果人工修订工具_自动打开.html”，这是正常现象。
- 如果产品第二次继续改，也可以直接把 step2_manual_review_package.json 放进“待修订输入”里，双击 BAT 后会优先自动加载这个修订包。
